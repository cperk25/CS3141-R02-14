####Sequence Diagram####
@startuml
actor Users as user
participant UI as ui
participant Buttons as b
participant Settings as set
participant "Main Menu" as mm
participant Pen as pen
participant Canvas as c

user -> ui: Interacts with
ui -> user: Timer, Win/Lose

b -> ui: Apart of
mm -> set: Effects

user -> b: Uses
b-> set: Sets Game setting (Free Draw or Game Mode)

user -> user: Plays: Single or Multiplayer

b -> pen: Set Color, Transparency, Size
b -> c: Download canvas

user -> pen: Uses tool
pen -> c: Draws on 

@enduml
